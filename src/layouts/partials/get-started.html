How to Get Started

Use one of the options above to download Bison Wallet. The latest release can always be found on GitHub.

For the installed version, open Bison Wallet using your system's application menu.

If you're running a binary from a terminal, open the displayed URL in the web browser of your choice.

Alternatively, launch DEX from within Decrediton, Decred's full-featured GUI wallet.

Detailed setup instructions are available on the wiki.

Start Bison Wallet or Decrediton. If using Decrediton, navigate to the DCRDEX interface.

Set a new "application password." Do not forget it!

If you already have a seed and are restoring your wallet, enter that seed now. Otherwise, a new seed will be generated for you.

The quick configuration dialog will offer you the option to intitialize built-in wallets and connect to known servers. If you aren't restoring from seed, you will be prompted to back up your newly generated application seed. Make sure to copy the seed carefully and put it somewhere safe. This seed is used to restore native wallets. Anyone with this seed has access to your funds.
Navigate to the settings view to connect to a DCRDEX access access trading markets. You can connect to a server and view market activity without cost, but if you want to trade, you'll need to post a bond

A bond is some amount of funds that will remain locked up on-chain for a while before you can retrieve them. This is an anti-spam measure to keep trades completing with fewer refunds.
Once you are bonded and you have the necessary wallet configured, you'll see an order form in the markets view. Set your price, quantity, and whether you are buying or selling, and then place an order. Orders will take between 0 and 15 seconds to start matching. Once you get matches, a settlement phase will begin which, depending on the assets and network conditions, may last only a few minutes, or last an hour or more in the worst case. The price you matched will never change.
